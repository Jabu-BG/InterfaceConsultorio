# Projeto: InterfaceConsultorio 

Sistema acadêmico desenvolvido em **Python** para simular um sistema interno de gestão de consultas médicas.

O projeto foi criado com o objetivo de praticar desenvolvimento em Python utilizando arquitetura organizada, banco de dados SQLite e interface gráfica com **Tkinter + ttk**.

O sistema representa o fluxo básico de trabalho de uma **secretaria hospitalar**, permitindo cadastrar pacientes, médicos e realizar o agendamento de consultas.

---

# Objetivos do Projeto

Este projeto tem como principais objetivos:

- Praticar desenvolvimento em **Python**
- Implementar operações **CRUD completas**
- Utilizar **SQLite** como banco de dados
- Desenvolver interface gráfica com **Tkinter + ttk**
- Aplicar **arquitetura em camadas**
- Implementar **regras de negócio reais**
- Utilizar a metodologia **RAD (Rapid Application Development)**

---
